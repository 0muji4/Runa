## アーキテクチャ

- **クライアント**: モバイルアプリ（Android / iOS）。
- **サーバ**: Go バックエンド（API・認証・データ保存など）。
- **インフラ**: マネージド DB・スケジュール実行などをマネージド部品に寄せ、自前で運用するのはアプリケーションのロジックに絞る。
